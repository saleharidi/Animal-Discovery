# Dog Breed Finder, Threatened Species, Bird Observation

A website composed of three different sites. The first is a dog breed finder. The second allows you to search about any species with detailed description and images. The third allows you to search a region for notable bird obserations.

### Technologies
* HTML
* CSS
* JavaScript

### Features 
* A hero section as a carousal, with random animal facts on display.
* A simple quiz to find the best dog for you.
* A section to display a random threatened animal.
